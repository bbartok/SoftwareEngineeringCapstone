# SE-OER Backend

## Initialize a virtual environment. This creates a folder called venv:<br />
virtualenv venv --python=python3<br />

### Activate the virtual environment:<br />
source venv/bin/activate<br />

### To exit the virtual environment:<br />
deactivate<br />
